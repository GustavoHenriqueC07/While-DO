# While-DO
Exercícios com do while em JavaScript

Este projeto reúne diversos exercícios desenvolvidos em JavaScript utilizando estruturas de repetição, principalmente o do while.

O objetivo foi praticar lógica de programação, validação de dados, contadores, condições e interação com o usuário utilizando prompt, alert e manipulação de HTML.

💻 Exercícios Desenvolvidos
1️⃣ Validação de Nome

O sistema solicita o nome do produto e continua pedindo até que o usuário digite um nome válido com pelo menos 3 caracteres.

Conceitos utilizados:
do while
validação de entrada
.length
2️⃣ Sistema de Senha

O usuário deve digitar a senha correta.
Enquanto a senha estiver incorreta, o programa continuará solicitando uma nova tentativa.

Conceitos utilizados:
comparação com !==
repetição
validação
3️⃣ Jogo da Adivinhação 🎯

O usuário deve adivinhar um número secreto entre 0 e 10.

O sistema:

conta quantas tentativas foram feitas;
informa quando o usuário erra;
exibe uma mensagem especial caso o usuário acerte de primeira.
Conceitos utilizados:
do while
contador
if/else
integração HTML + JavaScript
4️⃣ Tabuada

O usuário informa um número e o sistema gera a tabuada utilizando estrutura de repetição.

Conceitos utilizados:
for
concatenação
operadores matemáticos
5️⃣ Média de Notas

O usuário pode inserir várias notas até digitar um número negativo.
Ao final, o sistema calcula a média das notas digitadas.

Conceitos utilizados:
repetição
acumuladores
média aritmética
